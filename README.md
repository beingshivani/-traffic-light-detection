# 🚦 Traffic Light Detection

An AI-powered traffic light detection web app built with React and Claude Vision API.

## Features
- Upload any traffic scene image
- Detects Red / Yellow / Green signals using Claude AI
- Shows confidence scores, overlays, and detection logs
- Displays model performance metrics (mAP, Precision, Recall, F1)

## Tech Stack
- React (JSX)
- Anthropic Claude API (claude-sonnet-4-6 with vision)

## Dataset Reference
LISA Traffic Light Dataset + Roboflow. YOLOv8s architecture.
